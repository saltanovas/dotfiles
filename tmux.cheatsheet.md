1. control + b:

    - c: **Create** a new window

    - %: **Split** the current pane vertically
    - ": **Split** the current pane horizontally
    - o: **Switch** to the next pane
    - x: **Kill** the current pane

    - n: **Switch** to the next window
    - p: **Switch** to the previous window
    - &: **Close** the current window

    - d: **Detach** from the current session
    - s: **List** all sessions

2. Command prompts:
    - : **Enter** command mode to type tmux commands
    - rename-window: **Rename** the current window
    - rename-session: **Rename** the current session
    - new-session -s [name]: **Create** a new session with the specified name
        - or we can **detach** and just **tmux** again
